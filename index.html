We will be looking at an example of code in every section on at least one item explained.

This is an example of how the code will look like. Example from JavaScript.info
let name = "John";
embed a variable
alert( `Hello, ${name}!` ); // Hello, John!
embed an expression
alert( `the result is ${1 + 2}` ); // the result is 3
The expression inside ${…} is evaluated and the result becomes a part of the string.
We can put anything there: a variable like name or an arithmetical expression like 1 + 2 or something more complex.

Please note that this can only be done in backticks. Other quotes do not allow such embedding!

Data types

There are 7 data types that is commonly used in JavaScript.

For "Object and Symbol" I suggest you have a look at JavaScript.info.


Number - That is any number - 1, 2, 3, 4, 5, 6, 7, 8, 9, 0.

String - A string can have one or more characters. Anything you type between inverted commas ("") will be a string
It can be alphabetical, numerical or special characters.

Boolean – This is used to test for true or false.

Null - used for unknown values.

Undefined - used for unassigned values.

Object - for more complex data structures.

Symbol - for unique identifiers.

Code example - See JavaScript.info for more detail.

let myNumber = null; let myNumber = ""; In both cases the value that is assigned to "myNumber" is unknown and can be assigned later

Comparison operators

Most of the comparison operators was thought to us while attending school.

For the purpose of this document I will only list all the ones that is commonly used with one example of how to use it.

I suggest you have a look at JavaScript.info where it is explained in much more detail and a lot of examples.

Comparison operators are used to check “Boleen “ – true or fals

Operator Description Test Outcome
== Equal to 5==10 False
=== Equal to (and same type) 5==10 False
!= Not equal to 5!=10 True
!== Not Identical 10!==10 False
> Greater than 10>5 True
>= Greater than or equal 10>=5 True
< Less than 10<5 False
<= Less than or equal 10<=5 False

Code example - See JavaScript.info for more detail.

alert( 2 > 1 ); // true (correct)
alert( 2 == 1 ); // false (wrong)
alert( 2 != 1 ); // true (correct)

Assignment operators

For the purpose of this document I will only list all the ones that is commonly used with one example of how to use it.

I suggest you have a look at JavaScript.info where it is explained in much more detail and a lot of examples.

Operator Description Test Outcome
= Set equal to A = B A = B
+= Add and set equal to A += B A = A +B
-= Subtract and set equal to A -= B A = A - B
*= Multiply and set equal to A *= B A = A * B
/= Divide and set equal to A /= B A = A / B
%= Calculate remainder and set equal to A % B A = A % B

Code example - See JavaScript.info for more detail.

let a = 1;
let b = 2;

let c = 3 - (a = b + 1);
alert( a ); // 3
alert( c ); // 0
